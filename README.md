MyVuePress

VuePress 是一个基于 Vue 的轻量级静态网站生成器，以及为编写技术文档而优化的默认主题。 它是为了满足 Vue 自己的子项目文档的需求而创建的

[VuePress中文网](https://vuepress.docschina.org/)

### 搭建项目
1. 初始化项目
```
npm init -y
```
2. 安装vuePress
```
npm install vuepress -s
```
3. 在项目文件夹中新建相应的文件夹和文件
项目结构如下
```
MyVuePress
|————docs
|  |————README.md
|  |————.vuepress
|  |
```
